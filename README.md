# Alec Strassen Website Starter

## Struktur

- Startseite: `index.html`
- Design: `style.css`
- Chemie: `chemie/index.html`
- Physik: `physik/index.html`
- Physik/Gymi: `physik/gymi/index.html`
- Physik/FMS: `physik/fms/index.html`

## Hochladen

1. Dateien in deinen lokalen Ordner `Website` kopieren.
2. In PowerShell im Ordner `Website`:
   ```powershell
   git add .
   git commit -m "website struktur erstellt"
   git push
   ```
3. Danach öffnen:
   `https://alecstrassen.github.io/astrassen/`

## Wichtig

Da deine Seite unter `/astrassen/` läuft, wurden relative Links verwendet.
Keine Links mit `/` am Anfang verwenden.
